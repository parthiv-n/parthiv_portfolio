# Portfolio Website

A React-based portfolio website with a nature-inspired, organic aesthetic featuring weathered borders and mobile-responsive design.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React
- Vite
- Tailwind CSS
- Jeju Myeongjo font

## Features

- Mobile-responsive design
- Weathered border effects using SVG filters
- Four sections: Home, About, Leadership, Work
- Organic, nature-inspired aesthetic
